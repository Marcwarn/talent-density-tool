# Talent Density OS

Ett fristående Next.js-verktyg för ledningsgrupper som vill bedöma och höja talent density med fokus på:

- talent density score per ledare
- succession och bench strength
- retention/flight risk
- värdepassning och leadership impact
- prioriterad 90-dagars action board

## Kör lokalt

```bash
npm install
npm run dev
```

Öppna `http://localhost:3000`.

## Innehåll i MVP:n

- portföljöversikt för ledningsgruppen
- kalibreringsvy för samtliga ledare
- fördjupat individkort med styrkor, gaps och actions
- risk- och successionsvy
- prioriteringsboard för närmaste 90 dagarna
